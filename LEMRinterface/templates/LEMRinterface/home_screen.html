The goal of the Learning EMR project is to improve the display of EMR data.

For this study, you are to prepare for and present 18 patient cases as if you were presenting during morning rounds.

Start by reviewing the Evaluation Study Introduction slides.

Then, click the interface_demo button to practice using the interface. There will be three cases, one for each interface version.

Next, calibrate the eye tracking device. If the three red lights are on, press Ctrl + Shift + F10.

Finally, find the button with your participant code on it and click it to begin.
